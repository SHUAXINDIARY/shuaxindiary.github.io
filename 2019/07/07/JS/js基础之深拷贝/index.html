基本数据类型&复杂数据类型(引用数据类型)

概念

  • 在js种，有2中数据类型：一类是基本数据类型，一类是复杂数据类型；
  • 基本数据类型有:null,undefined,number,string,boolean,symbol(ES6)
  • 复杂数据类型:Object/arrary等

    区别

  • 两者的区别就是在内存中的存储位置不同
  • 前者在栈中存储数据
  • 后者在堆中存储数据，而栈中存储的是数据在堆中存储的位置(引用)
